# Projetos em C# 

Este repositório reúne vários exercícios e pequenos projetos feitos durante meu estudo da linguagem C#.  
Cada pasta contém um projeto separado, com seu próprio código e um README explicando o que o programa faz.

---

## 📁 Lista de Projetos

- Área, Perímetro e Diagonal do Retângulo
- Cálculo de Idades
- Números Ímpares
- Menor de Três Números
- Alturas
- Ordem Crescente

---

## 📌 Sobre os Exercícios

Os códigos foram feitos para praticar:

- Entrada e saída de dados
- Estruturas condicionais (`if / else`)
- Laços (`for`, `while`)
- Listas e coleções
- Operações matemáticas básicas
- Programação orientada a objetos
- Organização de código em C#

---

## 🎯 Objetivo

O objetivo deste repositório é acompanhar meu aprendizado da linguagem C# e servir como portfólio das atividades que venho desenvolvendo.
